# Kanban Board (MVP)

A minimal Kanban board built with vanilla JavaScript. Allows creating tasks, dragging them between columns, and rearranging columns.  

---

## Features

- Add tasks to any column  
- Drag and drop tasks between columns  
- Rearrange columns via drag-and-drop  
- Lightweight state management using `Map` objects  

---

## Known Issues

- Drag-and-drop may have edge-case bugs  
- Column/task movement can sometimes behave unexpectedly  
- Performance may degrade with a large number of tasks or columns  
- No persistence: data resets on page reload  
- Not all UI/UX edge cases are handled  

---

## Usage

1. Open `index.html` in a modern browser.  
2. Add tasks using the “Add task” button in any column.  
3. Drag tasks between columns or rearrange columns by dragging the header.  

---

> **Note:** This is a minimal viable product (MVP). It is intended for testing, learning, and experimentation. Production use is not recommended.
